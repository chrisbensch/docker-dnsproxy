# Docker image for [dnsproxy](https://github.com/adguardteam/dnsproxy)

A Docker image for [dnsproxy](https://github.com/adguardteam/dnsproxy), a simple DNS proxy with DoH, DoT, DoQ and DNSCrypt support.
